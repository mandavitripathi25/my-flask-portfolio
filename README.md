# Data Science Portfolio | Mandavi Tripathi

Welcome to my personal portfolio website! This is a modern, responsive web application built using Flask (Python) and customized with a vibrant, tech-focused HTML/CSS theme to showcase my data science projects, skills, and background.

## 🚀 Features
- **Responsive Design:** Fully optimized for desktops, tablets, and mobile screens.
- **Dynamic Content:** Powered by a Python Flask backend delivering seamless updates.
- **Vibrant UI/UX:** Built with a clean, colorful dark-mode aesthetic featuring gradient highlights.
- **Project Showcase:** Highlights key machine learning, analytics, and data engineering projects.

---

## 🛠️ Tech Stack & Tools
- **Backend:** Python, Flask
- **Frontend:** HTML5, CSS3, FontAwesome Icons, Google Fonts
- **Data Science Toolkit (Interests/Skills):** SQL, Pandas, NumPy, Scikit-Learn, XGBoost, Tableau

⚙️ How to Setup and Run Locally
Follow these quick steps to get the website running on your local machine:

1. Clone the Repository:-
        Bash
git clone [https://github.com/mandavitripathi25/my-flask-portfolio.git](https://github.com/mandavitripathi25/my-flask-portfolio.git)
cd my-flask-portfolio

2. Install Flask
Ensure you have Python installed, then install Flask via your terminal:

Bash
pip install flask

3. Launch the Web Application
Run the Python script to start the local server:

Bash
python app.py

4. View in Browser
Open your browser and navigate to:
http://127.0.0.1:5000/
